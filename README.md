# Brewco – App Support

Brewco is an iOS app for coffee lovers to save coffee recipes, track brewing parameters, and repeat the perfect brew.

## Support
If you need help, have questions, or want to report an issue, please contact us at:

abllogra@gmail.com

## Privacy
Brewco does not require user accounts and does not collect personal information directly.

The app uses Google AdMob to display ads. AdMob may collect device identifiers, approximate location, and usage data for advertising and analytics purposes. Data collected by AdMob is not linked to the user’s identity.

For more information, please refer to Google’s privacy policy:
https://policies.google.com/privacy

## About
Brewco is designed for personal use, with a focus on simplicity, consistency, and better coffee brewing.
